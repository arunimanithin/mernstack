# 📋 Simple Task Manager

A clean, interactive Task Manager built with **HTML**, **CSS**, and **JavaScript** — no frameworks or libraries required.

---

## ⚠️ Important: How to Open Correctly

> All 3 files **must be in the same folder**. Do NOT open `index.html` by itself from your Downloads — move all files to one folder first.

```
task-manager/        ← create this folder
├── index.html
├── style.css
└── script.js
```

Then open `index.html` from inside that folder.

---

## 📁 File Structure

| File | Purpose |
|---|---|
| `index.html` | Page structure & markup |
| `style.css` | All styling, layout & animations |
| `script.js` | App logic — add, complete, delete, render |
| `README.md` | Project documentation |

---

## 🚀 Features

| Feature | How |
|---|---|
| ➕ Add Task | Type in the input and click **Add Task** or press `Enter` |
| ✅ Complete Task | Click the circle — turns green, text gets strikethrough |
| 🗑️ Delete Task | Click the trash icon — task fades out smoothly |
| 🧹 Clear Completed | One click removes all finished tasks |
| 📊 Live Stats | Total / completed / remaining counts update instantly |

---

## ▶️ How to Run

1. Download all 4 files.
2. Place them all in the **same folder**.
3. Open `index.html` in any modern browser (Chrome, Firefox, Edge, Safari).

```bash
# Optional: run via a local dev server
npx serve .
# then visit http://localhost:3000
```
